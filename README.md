## 1 - Requirements

`make` - for the installer

`go` - for installing fetchfetch/executing fetchfetch

## 1 - Install

```
$ make install
```

or grab binary from [releases](https://github.com/NikonP/fetchfetch/releases)

## 2 - Update

```
$ make update
```

## 3 - Uninstall

```
$ make uninstall
```
